# Take Profit Flow Diagram

```
                    ENTRY
                      │
                      ▼
    ┌─────────────────────────────────────┐
    │           POSITION OPEN             │
    │         SL: -2% (HARD STOP)         │
    └─────────────────────────────────────┘
                      │
                      │ Price moves...
                      ▼
         ┌────────────┴────────────┐
         │                         │
    LOSS ▼                         ▼ PROFIT
         │                         │
    ┌────┴────┐               ┌────┴────┐
    │  -2%    │               │  +5%    │
    │ HIT SL  │               │  TP1    │
    └────┬────┘               └────┬────┘
         │                         │
         ▼                         ▼
    ┌─────────┐          ┌─────────────────────┐
    │  EXIT   │          │  CLOSE 30%          │
    │  100%   │          │  SL → BREAKEVEN     │
    │  LOSS   │          │  (Risk-free trade)  │
    └─────────┘          └─────────────────────┘
                                   │
                                   │ Price continues up...
                                   ▼
                              ┌────┴────┐
                              │  +10%   │
                              │  TP2    │
                              └────┬────┘
                                   │
                                   ▼
                    ┌──────────────────────────┐
                    │     CLOSE 50%            │
                    │  ACTIVATE 5% TRAILING    │
                    │  (20% position remains)  │
                    └──────────────────────────┘
                                   │
                                   │
              ┌────────────────────┴────────────────────┐
              │                                        │
              ▼                                        ▼
    ┌─────────────────┐                    ┌─────────────────┐
    │  TRAILING STOP  │                    │   LET IT RUN    │
    │    TRIGGERED    │                    │   TO THE MOON   │
    │                 │                    │                 │
    │  Price drops    │                    │  +20%, +50%,    │
    │  5% from peak   │                    │  +100%...       │
    └────────┬────────┘                    └────────┬────────┘
             │                                      │
             ▼                                      │
    ┌─────────────────┐                             │
    │  EXIT REMAINING │◄────────────────────────────┘
    │      20%        │   (Eventually trailing stop
    │                 │    catches the reversal)
    │  PROFIT LOCKED  │
    └─────────────────┘


═══════════════════════════════════════════════════════════════

                      EXAMPLE TRADE

═══════════════════════════════════════════════════════════════

Entry: $1.00
Position: 100 units ($100 value)

┌──────────┬──────────┬───────────┬──────────┬───────────────┐
│  STAGE   │  PRICE   │  ACTION   │ REMAINING│    PROFIT     │
├──────────┼──────────┼───────────┼──────────┼───────────────┤
│  Entry   │  $1.00   │   BUY     │   100%   │    $0.00      │
│          │          │  SL@$0.98 │          │               │
├──────────┼──────────┼───────────┼──────────┼───────────────┤
│  TP1     │  $1.05   │ SELL 30%  │    70%   │   +$1.50      │
│  (+5%)   │          │ SL→$1.00  │          │  (30 units)   │
├──────────┼──────────┼───────────┼──────────┼───────────────┤
│  TP2     │  $1.10   │ SELL 50%  │    20%   │   +$5.00      │
│  (+10%)  │          │ TRAILING  │          │  (50 units)   │
│          │          │  @5%      │          │               │
├──────────┼──────────┼───────────┼──────────┼───────────────┤
│  Peak    │  $1.25   │  (hold)   │    20%   │   (running)   │
│  (+25%)  │          │ trail@    │          │               │
│          │          │ $1.1875   │          │               │
├──────────┼──────────┼───────────┼──────────┼───────────────┤
│ Trailing │  $1.19   │ SELL 20%  │     0%   │   +$3.80      │
│  Hit     │          │  EXIT     │          │  (20 units)   │
└──────────┴──────────┴───────────┴──────────┴───────────────┘

TOTAL PROFIT: $1.50 + $5.00 + $3.80 = $10.30 (+10.3%)

  ┌─────────────────────────────────────────────────────────┐
  │                                                         │

  │                                                         │
  │   TP LEVEL 1:   +5%  → Close 30%, SL to breakeven      │
  │                                                         │
  │   TP LEVEL 2:   +10% → Close 50%, activate trailing    │
  │                                                         │
  │   TRAILING:     5% behind highest price                │

  └─────────────────────────────────────────────────────────┘
```
